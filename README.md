## Why Aglyphs?

- **Ultra-compact** вЂ” describe entire teams in < 60 chars
- **Text-native** вЂ” works everywhere (no images needed)
- **AI-native** вЂ” symbols for reflection, memory types, orchestration
- **Extensible** вЂ” easy to add your own glyphs
- **Git-friendly** вЂ” pure text, diffs beautifully

## Installation / Usage

Just copy the glyphs into your Markdown. For better rendering, use a font that supports mathematical symbols (most modern ones do).

**Recommended fonts:**
- JetBrains Mono
- Fira Code
- DejaVu Sans Mono
- Any Unicode-friendly editor font

For a live playground, see the [web demo](https://github.com/rb-thompson/aglyphs) (coming soon).
